# ①課題名 
リフティング記録アプリ

## ②課題内容（どんな作品か） 
サッカーのリフティングの練習記録を管理するWebアプリ。
日付、回数、動画を記録・保存・表示・削除可能。
講義で習ったPDOを使用したセキュアなデータベース操作とXSS対策を実装。
動画ファイルをサーバーに保存して管理します

## ⑤工夫した点・こだわった点  
ファイル管理: 動画ファイルのアップロード・保存・削除を自動的に管理、
データベース削除時に対応するファイルも削除

ユーザーインターフェース: 確認ダイアログで誤削除を防止
